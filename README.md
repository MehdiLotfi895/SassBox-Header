# SassBox Responsive Header

A modern, clean, and fully responsive navigation header built with **HTML5** and **CSS3**.

## Features
- **Fully Responsive:** Optimized for Desktop, Tablet, and Mobile screens using CSS Media Queries.
- **Pure CSS Mobile Menu:** Includes a functional hamburger menu for smaller screens using the "Checkbox Hack" (no JavaScript required).
- **Interactive UI:** Features smooth hover effects on the logo and navigation links.
- **User Status Indicator:** Includes a styled "Online" status badge for the user profile.

## Project Structure
- `index.html`: Contains the semantic HTML5 structure for the navigation bar and hidden mobile menu.
- `style/style.css`: Contains all styling, including CSS variables, layout configurations, and responsive breakpoints.
- `img/`: Directory for all project assets like icons and profile images.

## How to Run
Simply clone the repository and open the `index.html` file in any modern web browser.
